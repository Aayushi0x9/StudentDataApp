# StudentData App

### A Flutter-based Student Management App

StudentsData is a Flutter application designed to store and manage student details with a beautiful, intuitive UI/UX. The app allows users to easily add, edit, delete, and update student information, including their names, grades, standards, and profile pictures.

## Features

- **Add Student:** Easily add student details such as name, grade, standard (std), and profile picture.
- **Edit Student:** Edit any of the stored student information (name, grade, std, and profile picture).
- **Delete Student:** Remove student details with a simple delete functionality.
- **Beautiful UI/UX:** Designed with an elegant and user-friendly interface in a blue-grey color scheme for an enhanced user experience.
- **Profile Picture Management:** Add or change profile pictures for each student.
- **Responsive Design:** Optimized for mobile devices with a seamless and smooth interface.
